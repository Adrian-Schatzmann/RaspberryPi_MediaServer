Plex Media Server – Projekt

Unser Projekt besteht darin, einen Plex Media Server auf einem Raspberry Pi einzurichten. Damit entsteht eine zentrale Plattform für Filme, Serien und Musik, die im Heimnetzwerk oder von unterwegs zugänglich ist. Ziel ist eine einfache, kostengünstige und benutzerfreundliche Lösung für die private Medienverwaltung.

Raspberry Pi 4 Model B

Kompakter Einplatinencomputer für Projekte, Server und Medienanwendungen.

CPU
Quad-Core Cortex-A72 @ 1,5 GHz
RAM
2 / 4 / 8 GB LPDDR4
Grafik
VideoCore VI, H.265/4Kp60, H.264/1080p60
Anschlüsse
2× Micro-HDMI, 2× USB 3.0, 2× USB 2.0
Netzwerk
Gigabit Ethernet, WLAN ac (2,4/5 GHz), Bluetooth 5.0
Speicher
microSD (OS), externe USB-Laufwerke
Strom
USB-C, 5 V / 3 A

Fertiges Projekt

So sieht das fertiggebaute Projekt aus – mit den Teilen aus dem 3D-Drucker und dem Raspberry Pi im Inneren. Das kompakte Gehäuse schützt die Hardware und macht den Plex Media Server zu einer stabilen und praktischen Lösung.
